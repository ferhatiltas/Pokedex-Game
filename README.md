# Pokedex Game
Pokemonda bulunan tüm karakterler hakkında genel bilgi veren uygulama.


Karakterlerin isim, resim, ağırlık, boy, tür ve evrimleşme durumları gibi kriterleri hakkında bilgiler verilmiştir.
